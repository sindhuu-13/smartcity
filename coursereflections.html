Course Reflections

1. Kinds of Problems in Nature

Iteration, Recursion, and Backtracking are common approaches to solving problems in programming.

2. Space and Time Efficiency

Time Efficiency determines execution speed, while Space Efficiency measures memory usage. They are vital for optimal system performance.

Classes of Problems: P, NP, and orders of growth like O(1), O(n), and O(log n).

3. Design Principles

  • Divide and Conquer
  • Greedy Algorithms
  • Dynamic Programming
  • Backtracking

4. Hierarchical Data and Tree Structures

Trees like BST, AVL, and Tries optimize searches and data organization.

5. Array Query Algorithms

Efficient query algorithms like Prefix Sum, Segment Trees, and Fenwick Trees are applied in analytics and gaming.

6. Trees vs. Graphs

Trees: Hierarchical, acyclic. Applications: File systems.
Graphs: May contain cycles. Applications: Networks.

7. Sorting and Searching Algorithms

Sorting: Bubble, Merge, Quick Sort.
Searching: Linear, Binary Search.

Applications: E-commerce, database indexing.
